# SearchWave 🌊

**Project 2 — Software Engineering Bootcamp**

## Overview

SearchWave is a GIF search engine built with HTML, CSS, and JavaScript. It allows users to search for animated GIFs by entering a keyword, which triggers a request to the Giphy API. Results are displayed in a responsive custom grid layout that works on both desktop and mobile devices.

---

## How to Use

1. Open the website in your browser
2. Type any keyword in the search bar (e.g. `cats`, `funny`, `sports`)
3. Click the **Search** button or press **Enter**
4. Browse the GIF results displayed in the grid

---

## Features

- Keyword-based GIF search powered by the Giphy API
- Responsive design that adapts to desktop and mobile screens
- Custom CSS grid built with Flexbox
- Clean and modern dark UI with smooth hover animations
- Keyboard support (press Enter to search)
- Loading and error state handling

---

## Technologies Used

- HTML5
- CSS3 / SCSS (Sass)
- JavaScript (ES6+)
- [Giphy API](https://developers.giphy.com/)
- Google Fonts (Syne)
- GitHub Pages (hosting)

## Ideas for Future Improvement

1. **Pagination** — Add a "Load more" button to fetch additional results beyond the initial 20 GIFs
2. **Category filters** — Add preset buttons (Trending, Funny, Sports, etc.) so users can browse without typing
3. **Favorites** — Allow users to save their favorite GIFs locally using localStorage
4. **Copy to clipboard** — Add a button on each GIF to copy its URL with one click
5. **Dark / Light mode toggle** — Let users switch between dark and light themes
