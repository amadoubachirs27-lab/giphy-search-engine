User Story 1 — Search

As a user, I want to type a keyword in a search bar and click a button, so that I can find GIFs related to my topic.

Acceptance criteria :

There is a visible search input field and a submit button
Clicking the button triggers a request to the Giphy API
Results appear on the page after the search


User Story 2 — Browse results

As a user, I want to see the GIF results displayed in a grid layout, so that I can easily browse through multiple images at once.

Acceptance criteria :

GIFs are displayed in rows and columns using a custom CSS grid
The layout is clean and images are properly sized
At least several results are shown per search


User Story 3 — Mobile experience

As a user, I want the website to work well on my phone, so that I can search for GIFs from any device.

Acceptance criteria :

On small screens (320px and below), images stack in a single column
The navigation and header stack vertically on mobile
The search bar is usable on a small screen
